# 🎓 Students API

A RESTful API built with Golang for managing student records. This project was developed to learn and practice backend development concepts including HTTP servers, routing, request validation, configuration management, database integration, structured logging, and graceful server shutdown.

## 🚀 Features

* Create Student API
* JSON Request & Response Handling
* Input Validation using Validator
* SQLite Database Integration
* Configuration Management with YAML
* Structured Logging using slog
* Graceful Server Shutdown
* Clean Project Architecture
* Error Handling & Standardized Responses

## 🛠️ Tech Stack

* Golang
* net/http
* SQLite
* cleanenv
* go-playground/validator
* slog
